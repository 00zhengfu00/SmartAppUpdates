##开源 Android App 增量更新库

**包含以下内容**：
> * 服务器端生成差异包的工程：AppUpdate
> * 客户端使用的开源apk合并库：ApkPatchLibrary
> * 引用ApkPatchLibrary，实现增量更新的ApkPatchLibraryDemo
> * 旧版本的微博Android客户端，以及服务端生成的新旧微博差分包。
